# 概要
兵ズにおけるサーバ/クライアント間の通信プロトコルについて定義する。
サーバ/クライアントはTCP通信を行うものとする。
1度の通信は1行で行われ、内容は基本的にJSONで行われる。

# 送受信データ
送受信する1バイトのバイナリデータを1つの文字として解釈する。
1バイトデータの文字への対応はASCIIコード表に準ずる。

# 送受信1単位について
一度の送信では改行コードを含まない

```
[JSON STRING] '\n'
```
